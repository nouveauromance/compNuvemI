# Computação em Nuvem I
Este repositório é dedicado aos materiais, resumos e atividades desenvolvidos durante as aulas da disciplina.

## Objetivo das aulas:
Apender fundamentos da computação em nuvem, explorar serviços AWS como Amazon S3, EC2, RDS, VPC, desenvolver habilidades práticas em soluções de computação em nuvem, entender conceitos de networking, segurança e banco de dados.
